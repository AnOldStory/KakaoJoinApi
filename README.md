Kakao Join Api V1.0.0
===============================

Description
-----------

> 이 저장소는 모임에 대한 참여 관리를 할 수 있는 카카오톡봇 api 서버입니다.


Installation
------------

Node.js + express + mysql + sequlize를 사용하여 제작 되어져있습니다.

~~~
$ npm install

$ npm start
~~~
